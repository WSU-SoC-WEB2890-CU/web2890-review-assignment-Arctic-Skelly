Frequently asked questions, get your knowledge beforehand!

HTML (which stands for Hyper Text Markup Language) is the skeleton of a website, like text and buttons. Meanwhile CSS (which stands for Cascading Style Sheets) styles the website, giving it its color and layout!

Sass and Bootstrap are extensions of CSS that streamline the process and allow for extra customization!
